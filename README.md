# Nucleus – eWallet (Sitio Web Estático)

Sitio web estático desarrollado con HTML y CSS que simula la landing page de una aplicación de billetera digital (eWallet) llamada Nucleus.

El proyecto presenta un diseño moderno enfocado en productos fintech, con secciones informativas, testimonios, comisiones y funcionalidades del servicio, todo dentro de una estructura de una sola página `(index.html)`.

El diseño está basado en una arquitectura responsive y modular utilizando metodología BEM y buenas prácticas de CSS moderno.

---

## Tecnologías utilizadas

- HTML5
- CSS3
- Normalize.css
- Google Fonts (Poppins)
- Custom Properties (variables CSS)
- Flexbox
- CSS Grid
- Media Queries
- Metodología BEM

---

## Estructura del proyecto

```text
/
├── css/
│   ├── normalize.css # Normalización de estilos
│   └── styles.css    # Estilos principales del proyecto
├── img/              # Recursos gráficos (logo, ilustraciones, app UI)
└── index.html        # Página principal del sitio
```

---

## Secciones del sitio

- Header con navegación y presentación principal de la app
- Sección “¿Qué es?” explicando las funcionalidades de Nucleus
- Sección de seguridad del sistema y autenticación
- Sección de comisiones y costos de uso
- Testimoniales de usuarios
- Footer con navegación secundaria y branding

---

## Estilos y maquetación

El proyecto fue desarrollado siguiendo un enfoque **Mobile First**, priorizando la experiencia en dispositivos móviles y escalando progresivamente a pantallas más grandes mediante media queries.

Se aplicaron buenas prácticas de CSS moderno, destacando:

- **Metodología BEM** para mantener el código organizado, escalable y reutilizable
- **Custom Properties** para centralizar colores, fuentes y estilos globales
- **Flexbox** para alineación y distribución de elementos en secciones como navegación y grids simples
- **CSS Grid** para layouts más complejos como secciones principales y testimoniales
- **Media Queries** para adaptar el diseño a diferentes resoluciones
- Uso de **Normalize.css** para consistencia entre navegadores
